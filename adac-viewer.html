    View, validate and edit ADAC XML files online

    The ADACT ADAC Viewer is a free browser-based workspace for ADAC XML files used in civil infrastructure and as constructed asset handover workflows. Upload one or more files to validate supported schemas, add DXF reference drawings, inspect mapped assets, edit or merge working copies and export review outputs.

    Validate and repair

    Validate ADAC 5.0.1 and 6.0.0 XML against the supported schemas. Review readable errors, valid value guidance and suggested fixes, or preview high-confidence repairs without changing the original upload.

    Inspect and compare

    Explore point, line and polygon assets using grid, map or satellite backgrounds. Control XML, DXF and regional reference layers, inspect project and asset details, select multiple assets, and measure distances or areas.

    Edit a working copy

    Update schema-controlled attributes and coordinates, bulk edit compatible fields, review recalculation warnings, duplicate, split, join or delete assets, and use Undo or Redo before downloading edited XML or an ADAC report PDF.

    What is the ADACT ADAC XML Viewer?

    It is a free browser-based workspace for validating, mapping, reviewing and editing ADAC XML files. It combines supported schema checks, DXF reference drawings, mapped asset inspection, working-copy editing, reports and XML downloads.

    Can I edit an ADAC XML file in the viewer?

    Yes. Enter Edit mode to update supported attributes or geometry, bulk update compatible fields, use optional recalculations where offered, and duplicate, split, join or delete assets. Schema-valid changes update the working copy automatically.

    Does the viewer change my original XML file?

    No. The uploaded file remains unchanged. Repairs and edits are applied to a temporary browser working copy, which you can review, undo, redo and download as a separate XML file.

    Who is the ADAC Viewer for?

    It is designed for engineers, surveyors, asset managers, developers and project teams who need to inspect ADAC XML files before submission, review or handover.

    Does the ADAC Viewer replace certified validation?

    No. A successful schema check confirms supported XML structure and allowed values only. It does not certify engineering or survey accuracy, asset completeness, topology, receiver acceptance, or compliance with project requirements.
