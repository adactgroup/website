    Online ADAC XML viewer

    Open, inspect and review ADAC XML files online

    The ADACT ADAC Viewer is a free browser-based tool for reviewing ADAC XML files used in civil infrastructure and as constructed asset handover workflows. Upload one or more XML files to inspect mapped assets, layer groups, attributes, labels, geometry and basic file checks in a visual workspace.

    What the ADAC Viewer checks

    Review asset locations, point, line and polygon geometry, ADAC IDs, status, owner, notes, layer visibility, mapped labels and report-ready asset details before progressing files through project review or submission.

    Who uses the tool

    Engineers, surveyors, asset managers, developers and project teams can use the viewer to open ADAC XML files, compare mapped assets, inspect attributes and communicate issues quickly during handover review.

    ADAC XML review support

    The viewer assists with ADAC XML review and interpretation. It does not replace certified engineering validation, council or utility acceptance checks, survey verification, or project-specific submission requirements.

    What is the ADACT ADAC XML Viewer?

    It is a free online ADAC XML viewer for opening ADAC files and reviewing mapped civil infrastructure assets, layers, labels, attributes and basic file checks.

    Who is the ADAC Viewer for?

    It is designed for engineers, surveyors, asset managers, developers and project teams who need to inspect ADAC XML files before submission, review or handover.

    No. The viewer is an inspection and review aid only. Users should confirm ADAC XML data against council, utility, project and professional requirements before relying on it.
